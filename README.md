# TnkFactory Pub SDK (iOS)

Tnk Pub SDK는 Tnk의 광고 네트워크 상에서 광고앱 이나 매체앱을 개발하기 위하여 제공되는 통합된 SDK이며 아래의 기능들을 사용하실 수 있습니다.

* 전면광고(Interstitial Ad)
* 배너광고(Banner Ad)
* 네이티브 광고(Native Ad)
* 비디오 광고(Video Ad)
